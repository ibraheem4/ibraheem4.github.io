Facebook Insights provides developers with metrics about Applications, Pages and any website, even those that have not implemented Facebook platform.

While Google Analytics has robust capabilities for monitoring site usage, using Facebook Insights can provide reporting reducancy as well as an additional source of data that can help drive strategic decisions.

Many developers implement other types of analytics packages such as New Relic, Mixpanel, Piwik and Guag.es. It really just depends on the use case, but it is important to have an analytics system in place.
