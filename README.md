ibraheem4.github.io
===================

A [Jekyll](https://jekyllrb.com) site.

# Startup

    bundle install
    bundle update jekyll
    bundle exec jekyll build --incremental --watch
    bundle exec jekyll serve --incremental --watch

    bower install
    bower update

    npm install
    npm update
